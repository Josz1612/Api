Respuestas tipo para quiz de preparación
¿Qué es un recurso en REST?
Es una entidad identificable por una URL (ej: /products).
¿Qué método HTTP se usa para crear?
POST
¿Qué código HTTP indica éxito en una consulta?
200
¿Qué código indica recurso creado?
201
¿Qué significa stateless?
Que el servidor no guarda estado entre peticiones.
¿Qué formato de datos es común en APIs REST?
JSON
¿Qué pasa si pides un recurso inexistente?
404 Not Found
¿Qué método se usa para actualizar?
PUT
¿Qué método elimina?
DELETE
¿Por qué es importante el formato de error consistente?
Facilita el manejo de errores en el cliente.
